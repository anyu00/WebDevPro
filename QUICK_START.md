## 🐛 トラブルシューティング

### 問題: アプリが起動しない

**原因:** Live Server が起動していない
```
解決: index.html → 右クリック → "Open with Live Server"
```

### 問題: Firebase に接続できない

**原因:** Firebase Config が間違っている
```
解決: 
1. js/main.js を開く
2. firebaseConfig オブジェクトを確認
3. apiKey, databaseURL などが正しいか確認
4. https://firebase.google.com コンソールで確認
```

### 問題: データが保存されない

**原因:** Firebase Realtime Database が有効化されていない
```
解決:
1. Firebase コンソール → Database
2. "Realtime Database を作成" をクリック
3. ロケーション選択 (asia-northeast1 推奨)
4. セキュリティルール: テスト用 {"rules": {".read": true, ".write": true}}
```

### 問題: 日本語が文字化けしている

**原因:** ファイルエンコーディング が UTF-8 ではない
```
解決:
1. VS Code 右下 「UTF-8」 確認
2. 違う場合: 「UTF-8」 クリック → 再度 UTF-8 選択
```

### 問題: モバイルメニューが出ない

**原因:** ビューポートが 900px 以上
```
解決: デバイスツール で 400px など狭い幅に設定
```
